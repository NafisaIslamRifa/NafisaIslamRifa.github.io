Developed a GPT-like Large Language Model from scratch based on the book *Build a Large Language Model (From Scratch)* by Sebastian Raschka.

## Key Contributions

- Implemented transformer-based architecture from the ground up.
- Built tokenization pipeline and vocabulary processing system.
- Developed embedding layers and positional encoding modules.
- Implemented self-attention, feed-forward networks, and layer normalization.
- Trained an autoregressive language model for text generation tasks.
- Explored sampling strategies including greedy decoding and Top-k sampling.
- Gained hands-on experience in end-to-end LLM development and optimisation workflows.
