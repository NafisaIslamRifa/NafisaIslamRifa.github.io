Built a network intrusion detection system using machine learning techniques to classify malicious and normal network traffic using the augmented UNSW-NB15 dataset.

## Key Contributions

- Utilised the augmented UNSW-NB15 dataset for intrusion detection modelling.
- Applied feature engineering and preprocessing for robust performance.
- Trained and evaluated multiple machine learning models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - Support Vector Machine (SVM)
  - XGBoost
- Compared model performance across classification metrics.
- Achieved highest accuracy with the Random Forest model.
